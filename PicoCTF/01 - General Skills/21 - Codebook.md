# Solución
Para este reto se deben de descargar dos archivos, code.py y codebook.txt, ambos en la misma carpeta, esto se logra usando los siguientes comandos:
```bash
wget https://artifacts.picoctf.net/c/1/code.py
wget https://artifacts.picoctf.net/c/1/codebook.txt
```
Una vez que los hemos descargado, la descripción nos indica que ejecutemos el archivo code.py, de modo que usaremos la siguiente línea de código:
```bash
python3 code.py
```
